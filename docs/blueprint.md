# **App Name**: SkillMatch Pro

## Core Features:

- User Authentication & Authorization: Secure candidate, company, and admin signup/login with role-based access control.
- Company Task Management: Allow companies to create, view, update, and delete hiring tasks with details like title, description, and deadline.
- Candidate Task Browsing & Submission: Candidates can browse available tasks and submit their work, which can be file-based or link-based.
- Submission Review & Evaluation: Companies can review submitted work, assign scores, provide feedback, and update submission statuses.
- Candidate Performance Dashboard: Candidates can view the status of their submissions and evaluation results, including scores and feedback.
- Admin Panel: Admins can manage all users (candidates, companies), tasks, and overall platform data.
- AI-powered Task Description Assistant: An AI tool to assist companies in generating concise and clear task descriptions based on their requirements.

## Style Guidelines:

- Primary brand color: A deep, professional blue representing trustworthiness and expertise. HSL(220, 60%, 40%) -> #295CA8.
- Background color: A very light, subtle blue-gray that offers a clean, uncluttered canvas. HSL(220, 20%, 95%) -> #F4F6FA.
- Accent color: A vibrant sky blue to highlight calls to action and important information, providing visual contrast. HSL(190, 80%, 60%) -> #32C4EA.
- Headlines will use 'Space Grotesk', a sans-serif font with a modern, slightly tech-inspired feel. Body text will use 'Inter', a clean, readable sans-serif, ensuring clarity for longer content.
- Use clean, geometric icons with a solid fill style to maintain a modern and professional aesthetic.
- Utilize grid-based layouts for dashboards and task listings to ensure optimal organization and content readability across various screen sizes, promoting efficient information consumption.
- Subtle and functional transitions for navigation and data loading, designed to enhance user experience without causing distraction.
